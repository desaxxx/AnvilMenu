# AnvilMenu

## Adding it to your project

*Latest release tag*: [![](https://jitpack.io/v/desaxxx/AnvilMenu.svg)](https://jitpack.io/#desaxxx/AnvilMenu)

### Maven
```
<repository>
  <id>jitpack.io</id>
  <url>https://jitpack.io</url>
</repository>

<dependency>
  <groupId>com.github.desaxxx</groupId>
  <artifactId>AnvilMenu</artifactId>
  <version>LATEST.RELEASE.TAG</version>
</dependency>
 ```
### Gradle
```
dependencyResolutionManagement {
  repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
  repositories {
    mavenCentral()
    maven { url 'https://jitpack.io' }
  }
}

dependencies {
  implementation 'com.github.desaxxx:AnvilMenu:LATEST.RELEASE.TAG'
}
```
